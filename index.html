PAGE DETECTION WITH AI

This personal project was initially to satisfy my curiosity and check the functioning of simple neural networks for the pre-treatment of images.

For this I use an Adaline network to convert the pixels of an image to black and white. The result was so good that I thought about adding an implementation of a KNN network for edge recognition and corner extraction.

With this system you can find the image corners automatically and with an extra step, you could edit the image to focus only the content of the paper.
